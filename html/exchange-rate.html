Exchange Rate Viewer

Description

This project used public API data with 8-bit currency unicode sets.
Currency boxes below is to simply show the convertable currencies.
Created on March 2022. By Sung-Jun(Tony) Baek.

How to use

Select origin currency to desired currency you wish to convert.
Enter the origin money amount. After pressing submit, the result will be shown in the result box.

Additional development info

Exchange rate provided by Bank of Canada: https://www.bankofcanada.ca
